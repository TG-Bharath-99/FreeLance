🚀 Freelance Online

Freelance Online is a full-stack freelance marketplace that connects Clients and Developers. Clients can post projects, review applications, and collaborate with developers, while developers can browse projects, apply, and communicate with clients after being accepted.

🌐 Live Demo: https://frontend-delta-ten-49.vercel.app/

✨ Features
👨‍💻 Developer
Create an account and log in securely
Browse real projects posted by clients
Search and filter projects
Apply to projects
Track application status (Pending, Accepted, Rejected)
Chat with clients after acceptance
🧑‍💼 Client
Create an account and log in securely
Post new projects
Manage posted projects
Review developer applications
Accept or reject applications
Chat with accepted developers
Simulated payment workflow

🚀 Getting Started
Clone the Repository
git clone https://github.com/TG-Bharath-99/FreeLance.git
Install Dependencies

Frontend

cd frontend
npm install
npm run dev

Backend

cd backend
npm install
npm run dev
